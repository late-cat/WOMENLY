About the Model

  • Algorithm: Random Forest Classifier
  • Library: scikit-learn (Pipeline + SimpleImputer)
  • Dataset: Augmented PCOS dataset (2,000 samples, based on Kaggle dataset)
  • Training: 80/20 train-test split with 5-fold cross-validation
  • Purpose: Screening indicator only — not a clinical diagnostic tool
  • Note: Metrics shown reflect performance on the augmented training dataset
